💸 Currency Converter React
Современный и удобный конвертер валют, созданный на React. Позволяет мгновенно пересчитывать суммы между разными валютами с чистым интерфейсом.

📋 Описание
Это приложение для тех, кто ценит простоту и точность. Проект построен на компонентном подходе React, что обеспечивает быструю работу и отзывчивость интерфейса.

✨ Особенности
🎯 Мгновенный расчет — значения обновляются сразу при вводе.

🎨 Стильный дизайн — минималистичный интерфейс в розовых тонах.

📱 Адаптивность — корректно отображается на телефонах и компьютерах.

🔄 Легкость — проект очищен от лишних зависимостей и мусора.

📂 Архитектура
src/App.jsx — основная логика конвертации и состояние приложения.

src/App.css — стили и визуальное оформление.

src/main.jsx — точка входа в React-приложение.

public/ — статические ресурсы и иконки.

🚀 Запуск проекта
Склонируйте репозиторий: https://github.com/Sezim673/currency-converter-react

Установите зависимости: npm install

Запустите проект: npm run dev

🛠️ Технологии
React — создание интерфейса.

Vite — быстрая сборка проекта.

JavaScript (ES6+) — логика вычислений.

CSS3 — кастомные стили.
